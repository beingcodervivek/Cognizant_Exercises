Exercise 19.1 & 19.2 - Sass Customization

Use npm to install Bootstrap: npm install bootstrap

Edit _variables.scss to change colors or border-radius:

$primary: #ff5722;
$border-radius: 1rem;

Then import and compile using Sass:

@import "node_modules/bootstrap/scss/bootstrap";
